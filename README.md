# 🤖 Vaani AI - Smart Virtual Assistant

Vaani AI is a smart virtual assistant that interacts using **text and voice**, performs tasks, and responds using **AI-powered intelligence**.

---

## Features

* User Authentication (Login/Register)
* AI Chat (OpenAI integration)
* Voice Input (Speech Recognition)
* Voice Output (Text-to-Speech)
* Real-time Weather Updates
* Google Search Integration
* Contact Form (MongoDB storage)
* Open system apps (Windows only)

---

**Frontend:**

## ⚙️ Installation & Setup

### 1. Clone Repository

```
git clone https://github.com/your-username/vaani-ai.git
cd vaani-ai
```

### 2. Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Create `.env` File

```
OPENAI_API_KEY=your_openai_key
SECRET_KEY=your_secret_key
MONGO_URI=your_mongodb_connection
```

### 5. Run Application

```
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```

---

## Environment Variables

| Variable       | Description               |
| -------------- | ------------------------- |
| OPENAI_API_KEY | OpenAI API Key            |
| SECRET_KEY     | Flask Session Key         |
| MONGO_URI      | MongoDB Connection String |
